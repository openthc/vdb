Cannabis Strain Database

3,000+ verified strains, 6000+ alternatives & suggestions

The Problem

There are over 100 software providers across the world dealing with cannabis data. Strain names, across the industry, would be nice to know, or at least keep consistent.

There are other offerings for a Strain API however, each has a unique, un-predictable numbering scheme. Others were in existance but have been removed, or disabled.

A Solution Proposal

A simple search API and static data-sets in multiple formats for any/all software providers to use.

A clearly defined normaliztion pattern.

A community driven submission and verification system.

An open license on the data to keep it available for everyone.
